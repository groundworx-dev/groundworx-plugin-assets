Groundworx Foundation – Changelog

1.1.0

  • Added Responsive Grid block with per-breakpoint grid/flex layout, column, alignment, and equal-heights controls
  • Added Core Image Extend support for per-breakpoint image overflow, max-width, and justification
  • Added Abbreviation rich text format
  • Added Gradient Text rich text format
  • Rebuilt Form Base settings panel with expanded controls
  • Overhauled license activation and plugin updater system
  • Redesigned admin settings page
  • Added responsive utilities for breakpoint cascade resolution
  • Refactored plugin entry points to a central index.js / main.js pattern for better performance
  • Webpack now auto-discovers support entry points
  • Removed responsive-content-size support module

1.0.2

  • Added translation support with POT file for localization

1.0.1

  • Fixed text domain to use groundworx-foundation consistently across all files
  • Added Text Domain header to plugin file

1.0.0

  • Initial release
  • Core block extensions for Columns, Column, Group, List, and Headings
  • Responsive layout controls
  • Global form styling system
  • Settings panel for module management
