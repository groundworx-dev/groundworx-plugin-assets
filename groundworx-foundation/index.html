Groundworx Foundation – Changelog

Version 1.0.0 – January 2025

  • Initial release.
  • Added core block extensions for Columns, Column, Group, List, and Headings.
  • Responsive layout controls per breakpoint.
  • Full Height support for viewport-height sections.
  • Has Background Padding for automatic padding when background color is set.
  • Responsive Content Size and Text Align controls.
  • Global form styling system for inputs, selects, checkboxes, and radios.
  • Settings panel for module management at Tools > Groundworx > Foundation.
