Groundworx Query Filters – Changelog

Version 1.0.0 – January 2025

  • Initial release.
  • Added Filters container block with Query Loop connection.
  • Added Filter Terms block with dropdown, checkbox, and radio display options.
  • Added Filter Text block for keyword search.
  • Added Filter Orderby block for sort controls.
  • Added Filter Submit block for manual filter application.
  • Added Filter Active block to display current filters with remove buttons.
  • URL parameter sync for shareable filtered views.
  • Taxonomy term counts and hierarchical display support.
