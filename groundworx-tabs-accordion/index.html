Groundworx Tabs & Accordion – Changelog

Version 1.0.0 – January 2025

  • Initial release.
  • Added Tabs and Tabs Panel blocks with horizontal and vertical layouts.
  • Added Accordion and Accordion Panel blocks with single or multiple open modes.
  • RTL (right-to-left) support with dedicated variations.
  • Keyboard navigation (arrow keys, Home, End) and ARIA attributes for accessibility.
  • Icon customization options (style, background, position).
  • Tab alignment options: left, center, right, and fill.
  • Smooth open/close animations for accordion panels.
  • Built with WordPress Interactivity API – no jQuery dependencies.
