Groundworx Tabs & Accordion – Changelog

1.2.0

Accessibility (ADA Compliance)

  • Added visible focus outlines on all trigger buttons
  • Added reduced motion support — animations disabled when preferred
  • Roving tabindex on tab buttons
  • Dynamic tab panel roles for responsive tabs-to-accordion switching
  • Proper accordion ARIA pattern with context-driven labels
  • Closed panels use inert attribute to prevent keyboard access
  • Cleaned up ARIA attributes on non-interactive elements

Keyboard Navigation

  • Refactored to use Interactivity API context instead of DOM queries
  • Reactive focus management via data-wp-watch callbacks
  • Auto-activate on arrow keys only in tabs display mode
  • Focus transfers to correct trigger on responsive layout switch

SEO

  • Added FAQ and How To structured data (JSON-LD) output
  • Schema type selector in inspector: Off / FAQ / How To

New Inspector Controls

  • Accessible Name field for screen reader labels
  • Structured Data type selector in Accessibility & SEO panel

Security

  • Typography inline style values now properly escaped

General

  • Updated admin settings and licensing
  • Improved build system and asset loading
  • Removed unused files

1.1.2

  • Fixed editor panel content padding by adding missing `gwx-panel` wrapper to match frontend DOM structure

1.1.1

  • Fixed default inactive trigger background color on accordion panels

1.1.0

  • Overhauled color application for triggers and panels using Interactivity API directives
  • Added extend-trigger-to-panel option to apply active colors across the full panel or just the trigger
  • Fixed RTL icon flipping to apply at block level instead of only accordion mode
  • Fixed multi-accordion panel state management

1.0.1

  • Fixed text domain to use groundworx-tabs-accordion consistently across all files
  • Added Text Domain header to plugin file

1.0.0

  • Initial release
  • Tabs and Tabs Panel blocks with horizontal/vertical layouts
  • Accordion and Accordion Panel blocks
  • RTL support and variations
  • Icon customization options
  • WordPress Interactivity API integration
