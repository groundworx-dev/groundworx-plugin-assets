Groundworx Showcase – Changelog

2.0.0

  • Migrated carousel frontend from vanilla JS to WordPress Interactivity API (view-flow.js), removed embla-init.js
  • Full breakpoint-responsive mode switching: carousel modes reinit Embla, flex/grid modes destroy Embla and switch to CSS layout
  • Reactive CSS custom property bindings (data-wp-style) for slide size, gap, columns, and fade mode class toggling across breakpoints
  • Added carousel effects support (scale, opacity, grayscale) via Interactivity API
  • Dynamic Flow block now uses the shared Interactivity API store (matching Static Flow)
  • Refactored shared PHP helpers into inc/flow-helpers.php for Static Flow and Dynamic Flow
  • Rewrote Dynamic Content view.js to follow the same Interactivity API patterns as view-flow.js
  • Added large-phone (480px) breakpoint to editor UI for all blocks
  • Added Add Slide toolbar button on individual Flow Slide blocks (inserts after current slide)
  • Added cross-transforms between all pagination, counter, and progress blocks
  • Fixed editor crash when duplicating multiple Flow Slides at once (debounced context propagation)
  • Fixed Duration control showing 15ms instead of Inherit on breakpoint tabs
